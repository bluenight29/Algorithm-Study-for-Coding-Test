### 스터디 참여하는 방법

1. 알고리즘을 이해한다.
2. 책에 제공된 Python 코드를 이해한다.
3. C언어로 코드를 짜본다.
4. 파일명은 `문제번호_작성자.c`로 한다.
5. PR(Pull Request)한다.
6. Commit message: `문제이름`으로 통일해주세요. (ex: `거스름돈 문제`)
